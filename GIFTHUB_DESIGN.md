# Design Document: Gifthub - POD Gift Shop Redesign

## 1. Brand Identity: Gifthub
- **Name:** Gifthub
- **Domain:** gifthub.com
- **Core Value:** Personalization (Custom Design) for meaningful gifts (Family, Friends, Pets).
- **Vibe:** Warm, Friendly, Trustworthy, Creative.

## 2. Visual Style Guide
- **Color Palette (Warm & Friendly):**
  - **Primary:** Coral/Soft Red (`#FF6B6B`) - Symbolizing love and gifts.
  - **Secondary:** Soft Cream/Off-white (`#FFF9F5`) - For a warm background.
  - **Accent:** Warm Gold (`#FFD93D`) - For highlights and quality badges.
  - **Text:** Deep Charcoal (`#2D3436`) - For readability.
- **Typography (Minimalist & Modern):**
  - Primary Font: **Inter** or **Geist** (already in project) for a clean, modern look.
  - Heading Font: Bold versions of the primary font for clarity and impact.

## 3. Key UI Components Redesign
### A. Header
- **Logo:** "Gift" (Coral) + "hub" (Dark) with a small gift icon.
- **Navigation:** Browse by "Recipients" (Family, Friends, Pets) and "Occasions" (Birthday, Anniversary).
- **Cart & Account:** Minimalist icons.

### B. Product Page (Main Landing)
- **Hero Section:** Highlight the "Personalized" aspect with clear call-to-actions.
- **Customization Form:** Simplify the UI, use Coral for buttons and active states.
- **Preview:** Ensure the "TemplatePreview" looks prominent and high-quality.
- **Social Proof:** Update "Popular" badges and star ratings to match the theme.

### C. Footer
- Focused on trust: Shipping info, Satisfaction guarantee, Social links.
- Brand story: "Gifthub - Where every gift tells a story."

## 4. Technical Updates
- Update `globals.css` with new CSS variables.
- Refactor `Header.tsx`, `Footer.tsx`, and `app/page.tsx`.
- Update branding across all metadata and text content.
